# Android S24 Gate 3 Evidence

`gate3-pass-summary.json`은 `android-v0.1.0-gate3-pass` 태그의 Samsung S24 Ultra 정확 프레임 Gate 3 결과를 비식별 정보만 남겨 정리한 증거다.

- 원본 실기기 보고서는 로컬 `android/reports/s24-frame-accuracy-report.json`이며 Git에 포함하지 않는다.
- 앱·계측 APK도 Git에 포함하지 않고 SHA-256만 보존한다.
- ADB serial, Windows 사용자 경로, 실제 사용자 영상과 파일 경로는 기록하지 않는다.
- latency와 memory 값은 당시 측정 snapshot이며 제품 성능 합격선이 아니다.
